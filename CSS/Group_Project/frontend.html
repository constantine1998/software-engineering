SecureSend

Send and receive files securely with Cryptographic Technique


There will be two methods available for file transfer:

  • Cryptographic
  • Non-Cryptographic

Both of these methods use Peer to Peer sharing


Using Cryptographic method

  • Send and receive files securely and safely
  • This is a little slower than Non-Cryptographic method

Using Non-Cryptographic method

  • Send and receive files safely but not securely
  • This is faster than Cryptographic method
